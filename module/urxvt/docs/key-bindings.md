We define key bindings for URxvt in [`config/urxvt.xresources`](../config/urxvt.xresources).

Copy and paste

- `CTRL+ALT+c` : Copy
- `CTRL+ALT+v` : Paste

Resize font

- `CTRL++` : Increase font size
- `CTRL+-` : Decrease font size
- `CTRL+=` : Reset font size to default
- `CTRL+?` : See current font settings
